vjvj_crawler
============

##Requirements

Python 3+ 

scrapy
```
  pip install scrapy
```

##Run Setting

config directory 의 config.cfg 에 크롤링할 정보 입력  

```
  >>> main.py 섹션명
```

##Run

```
  >>> main.py
```

##config.cfg 구성

```
[섹션명1]
키1=값
키2=값
키3=값
[섹션명2]
키1=값            # 섹션이 다르면 키값 동일 구성가능
키4=값
키5=값
```
